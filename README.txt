Project Title:
AI Digital Twin for Smart Campus Energy Monitoring

Description:
This project predicts campus electricity usage using machine learning.
A Linear Regression model is trained using student population and temperature
data. The results are displayed through a Streamlit dashboard.

Tools Used:
Python
Pandas
Scikit-learn
Streamlit

Files:
model.py – machine learning model
app.py – dashboard interface
campus_data.csv – dataset
